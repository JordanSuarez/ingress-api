# ingress-api

## Requirements
- Docker & Docker compose

## Install process
- `make build`
- `make run`
- `make connect` 
- `make composer-install`
- `make create-db`
- `make migration`
- `make migrate`
- `make fixtures`

## Phpmyadmin

`http://localhost:8080`

## Api base url

`http://localhost:8741`
